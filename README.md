# Twitter Archive MCP (TypeScript)

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Build the project:
   ```sh
   npm run build
   ```
3. Start the server:
   ```sh
   npm start
   ```

## Scripts
- `npm run build` — Compile TypeScript to JavaScript
- `npm start` — Run the compiled server

## Project Structure
- `src/` — TypeScript source files
- `dist/` — Compiled JavaScript output
